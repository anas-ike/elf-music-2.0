## 2. Genius

**Endpoint:** `/genius/lyrics`  
**Method:** `GET`

**Parameters:**
- `title` (string) ‚Äì The song title.
- `api_key` (string) ‚Äì Your Genius API Key.

**Example Request:**

**Example Response:**
